# Portafolio - Desarrollo y Operaciones de Software

Este repositorio contiene los trabajos, laboratorios y proyectos desarrollados durante el curso de Desarrollo y Operaciones de Software.

A continuación se detalla el contenido y lo que se realizó en cada uno de los directorios:

## 1. [Lab 3](./Lab%203/)
**Tema:** TDD – Pruebas de Software – Agilismo y Scrum – Análisis de Requerimientos.
**Descripción:** 
- Identificación de reglas de negocio para un sistema de gestión de cuentas bancarias.
- Elaboración de diagramas de contexto, casos de uso, historias de usuario (aplicando atributos de calidad) y diagrama de clases.
- Implementación de la técnica Planning Poker (serie de Fibonacci) para estimación de esfuerzo en historias de usuario.
- Desarrollo del sistema bancario aplicando principios SOLID (Single Responsibility, Open/Closed) y patrones de diseño (Strategy, Factory Method).
- Creación de pruebas unitarias usando JUnit, medición de cobertura de código con JaCoCo (superando el 80%), e integración de análisis estático con SonarQube mediante Docker.

## 2. [lab 2](./lab%202/)
**Tema:** Laboratorio 2 CVDS-DOSW-01
**Descripción:** 
- Proyecto de demostración y configuración base usando Spring Boot y Maven.
- Práctica inicial de estructura de un proyecto Spring Boot.

## 3. [Parcial_corte_1-2025-2](./Parcial_corte_1-2025-2/)
**Tema:** Examen Parcial - Primer Corte 2025-2
**Descripción:** 
- Resolución de un caso de estudio mediante diagramas de contexto, casos de uso y diagrama de clases.
- Aplicación de principios SOLID (SRP, OCP, LSP).
- Implementación de patrones de diseño Creacional (Factory Method) y Estructural (Bridge) orientados a un sistema de reportes.
- Desarrollo del código en Java e implementación de pruebas unitarias para garantizar el funcionamiento.
- Verificación de métricas de calidad de software usando JaCoCo y SonarQube.

## 4. [Taller_Evaluativo](./Taller_Evaluativo/)
**Tema:** Taller Evaluativo 2025-2 - Sistema de Inventario
**Descripción:** 
- Desarrollo de una aplicación Maven + Spring Boot orientada a la gestión de un inventario y notificaciones.
- Resolución de requerimientos respetando estrictamente los principios SOLID.
- Implementación del Patrón de Diseño Observer para notificar cambios de stock a distintos agentes (AgenteLog, AgenteAdvertencia).
- Planificación del trabajo en Épicas, Features e Historias de Usuario.
- Desarrollo impulsado por pruebas (JUnit 5) con un mínimo del 80% de cobertura en JaCoCo y evaluación estática con SonarQube.

## 5. [Taller_Evaluativo_T2](./Taller_Evaluativo_T2/)
**Tema:** Taller Evaluativo 2 2025-2 - API REST Master Chef Celebrity
**Descripción:** 
- Desarrollo de una API RESTful en Spring Boot para gestionar recetas de un programa de cocina (Televidentes, Participantes, Chefs).
- Implementación de controladores, servicios y modelos aplicando el patrón de diseño Builder y Factory.
- Integración de persistencia con base de datos NoSQL usando MongoDB.
- Configuración de un pipeline CI/CD empleando GitHub Actions para despliegue automatizado hacia Microsoft Azure.
- Documentación y prueba de los endpoints a través de Swagger.
- Mantenimiento de calidad del código evidenciado con métricas de JaCoCo y SonarQube.
